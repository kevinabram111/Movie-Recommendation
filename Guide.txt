Guide:

How to run the iOS App:
1. Install Xcode First
2. Navigate to the iOS App and open movie-recommendation.xcodeproj
3. Wait for the load to end, then tap Run

How to run the Backend:
1. Install Python
2. Navigate to the directory "Backend and Machine Learning" first
3. Go to the terminal and install surprise: pip install scikit-surprise
4. In that directory in the terminal, just type flask --flask_project run to run the app
5. The local server will run locally. It will usually be http://127.0.0.1:5000
6. You can then try out the endpoints, for example, http://127.0.0.1:5000/user/1/movies

How to run the Rmd?
1. Install RStudio first
2. Open the Rmd file in R Studio that is installed
3. Make sure you have a TMDB account: https://developer.themoviedb.org/reference/intro/getting-started
4. Input the bearer token you have, then you can run one by one
